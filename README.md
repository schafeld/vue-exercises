## Course

This repo was started following Coursera's [The Complete Vue.js Course for Beginners](https://www.coursera.org/learn/packt-the-complete-vue-js-course-for-beginners-i44iy) (Packt).

The course seems outdated and very basic, abandoned and switched to other course(s).

For this repo the [Vue 3 installation](https://vuejs.org/guide/quick-start.html) official guide was used for getting set up.
This repository's code uses the composition API (not the options API).

Switched to [Vue.js 3 for Beginners: Build Modern Web Applications](https://www.coursera.org/learn/packt-vue-js-3-for-beginners-build-modern-web-applications) that uses Pinia and composition API and Vue 3 explicitely.
